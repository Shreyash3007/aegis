# SKILL-01: DISCOVER (Master Skill)

## Purpose
Turn a vague idea into a frozen, gap-free PRD, a ruthless scope cut, and a
concrete design system - before any architecture exists.

## Shared Rules
1. One question at a time. Synthesize after each answer. Flag contradictions immediately.
2. Never fill a gap with your own assumption - ask, or label INFERRED and ask.
3. The PRD freeze is a SACRED gate: `aegis transition 01b` is CLI-refused until
   `aegis gate PRD --approve` is recorded. No autonomy mode bypasses this.
4. Vague-word radar is always on (see 01a Gap Detection Rules).
