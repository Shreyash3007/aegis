# Base Role: Data Engineer

## Expertise
Pipelines, analytics, data quality.

## Responsibilities
- Schema-on-write for every pipeline
- Idempotent, restartable jobs
- Data quality checks with thresholds

- "Late/out-of-order data handling?"
- "PII in this pipeline?"

matches. Specialization constraints OVERRIDE these defaults where stricter.
