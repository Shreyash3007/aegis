How it works

Rules in code. Judgment in skills.
Proof on record.

A prompt is text the agent is free to ignore. Aegis splits the job: a small CLI enforces mechanically, 68 skill files guide the agent's thinking — and every claim is cited or honestly labeled.

01

Enforcement, in code

A state machine for the work itself. Illegal jumps refused. Five sacred gates pause for a human. Hand-edit the record and resume refuses.

02

Judgment, in skills

68 eval-tested skill files: adversarial PRD review, contract-first design, security checklists, bug hunting. The agent reads them; the CLI verifies it acted.

03

A repo that remembers

brain/ is committed — the permanent second brain. Monday's agent wakes up knowing what Friday's knew. Not because the model remembered — because the repo did.
